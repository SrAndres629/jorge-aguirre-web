# Documentación de Flujos de Automatización

Este directorio contiene la documentación y definiciones de los workflows de n8n para el ecosistema "Jorge Aguirre Ecosistema 360º".

## Estructura de Workflows
1. **Inteligencia de Marketing**: Meta Bridge (CAPI)
2. **Admisión**: Triage enriquecido (Sofía)
3. **Comercial**: Cierre contextual
4. **Operaciones**: Agenda
5. **Éxito**: Fidelización
